# Summary Template

Default summary filename:

```text
paper_summaries/<PaperShortName>_摘要.md
```

Recommended structure:

```markdown
# <论文标题>

- 作者与年份：
- 主要问题：
- 核心方法：
- 主要贡献：
- 与当前 metric indexing proposal 的关系：
- 可以支撑 proposal 的哪些段落：
- 仍需回原文确认的点：
```

Writing guidance:

- Prefer concrete claims over vague praise.
- State whether the paper supports background, related work, method motivation, or evaluation design.
- If the paper is not directly useful, say so explicitly and explain why.
- Keep the summary readable enough that later proposal drafting can start from this file without reopening the PDF.
